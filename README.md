## ✨ What is this?

A complete, **free, open-source** roadmap for backend developers who want to become **production-ready AI Engineers** in 12 months — without quitting their job.

No fluff. No academic rabbit holes. Just the skills the industry demands, in the order you need to learn them, with real projects to prove your knowledge.

> **Who is this for?**  
> Backend/full-stack developers (Node.js, Python, Java, Go) with 1–3+ years of experience who want to transition into AI Engineering.

## 🖥️ Run Locally in 60 Seconds

No installation required. No Node. No Python. No build step. Just one file.

```bash

# Open the roadmap
cd ai-engineer-roadmap
open index.html          # macOS
```

Or just **double-click `index.html`** in your file explorer. Works entirely offline in any modern browser.

## 🔥 2026 AI Engineer Skill Coverage

- **Core LLM** — APIs, streaming, cost tracking, rate limiting, multi-provider
- **Prompting** — zero-shot, CoT, versioning, A/B testing, structured output
- **Embeddings** — chunking strategies, vector DBs, hybrid search, reranking
- **RAG** — advanced RAG, RAGAS evaluation, Bedrock Knowledge Bases, GraphRAG
- **Agents** — ReAct from scratch, tool design, multi-agent, human-in-the-loop
- **MCP** — server design, authentication, npm publishing, Claude Desktop
- **Frameworks** — LangChain LCEL, LlamaIndex, LangSmith production tracing
- **Evaluation** — evals as code, golden datasets, CI regression gates
- **Testing** — LLM mocking, adversarial red-teaming, load testing, CI/CD
- **AWS** — Bedrock, SageMaker, VPC-private deployments, full IaC
- **Production** — semantic caching, model routing, circuit breakers, auto-scaling
- **Observability** — OpenTelemetry, cost dashboards, quality monitoring
- **Security** — prompt injection defense, PII redaction, OWASP LLM Top 10
- **Fine-tuning** — LoRA/QLoRA, OpenAI API, HuggingFace PEFT
- **Local LLMs** — Ollama, vLLM, quantization, air-gapped deployment
- **System Design** — capacity estimation, multi-tenant design, case studies
