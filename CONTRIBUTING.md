# Contributing to AI Engineer Roadmap

Thank you for helping make this roadmap better! This project is maintained by the community.

## What to contribute

- **Fix outdated content** — library versions, deprecated APIs, changed pricing
- **Improve exercises** — better hands-on exercises for any phase
- **Add resources** — genuinely useful links (no spam, no affiliate links)
- **Fix bugs in the UI** — the roadmap is all vanilla HTML/CSS/JS in `index.html`
- **Share your experience** — open a Discussion about how long phases actually took

## How to contribute

1. Fork the repo
2. Make your changes in `index.html` or `README.md`
3. Test `index.html` by opening it locally in a browser
4. Open a PR with a clear description of what you changed and why

## Code style for index.html

- All logic lives in `<script>` at the bottom of `index.html`
- Data is in the `PHASES`, `WEEKS_DATA`, `PROJECTS_DATA`, `CHECKLIST_DATA`, and `INTERVIEW_DATA` arrays
- CSS variables are used for all colors — dark mode works automatically
- No external dependencies — everything must work offline

## Reporting issues

Open a GitHub Issue with:
- Which phase or week has the problem
- What's outdated or incorrect
- What the correct information should be (with a source if possible)

Thank you for contributing! ⭐
